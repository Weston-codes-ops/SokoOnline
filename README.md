OVERVIEW OF SOKOONLINE
SokoOnline is a production-ready online marketplace built from scratch, featuring a Spring Boot REST API backend and a React frontend. 
The platform supports the full e-commerce flow — from browsing and filtering products, to cart management, checkout, and order tracking.
